# 07. Cost Analysis — 규모별 비용 시뮬레이션, 손익분기점 분석

## 설계 기준

모든 비용은 **ap-northeast-2 (서울) 리전**, 2024년 기준 AWS 공시 단가 기반입니다.
핵심 원칙: **유휴 비용 최소화, 실행 시에만 과금, 규모에 따라 인프라 자동 전환.**

### 손익분기점

```
비용 ($)
  │
  │                                        ╱ Lambda + DynamoDB
800├─────────────────────────────────────╱──
  │                                  ╱
600├──────────────────────────────╱─────
  │                           ╱
400├─────────────────── ─ ─ ╳─ ─ ─ ─ ─── ECS Fargate ($360 고정)
  │                     ╱ ↑
200├──────────────────╱── 손익분기: ~1.5억 건/월
  │               ╱
  0├──╱──────────────────────────────────
   0    5000만    1억    1.5억    2억    3억   (월 호출 수)
```

**손익분기점: 월 ~1.5억 건**
- 이하: Lambda가 저렴 (서버리스, 유휴 $0)
- 이상: ECS가 저렴 (고정 비용 안에서 무제한)

## On-Prem vs AWS 비용 비교

| 항목 | On-Prem (현재 추정) | AWS 시나리오 1 | AWS 시나리오 2 | AWS 시나리오 3 |
|------|-------------------|-------------|-------------|-------------|
| 규모 | 개발/검증 | 중소형 | 대형 | 메가 |
| 서버 | GPU 서버 $2,000+/월 | - | - | - |
| Airflow | EC2 $100+/월 | $0 | $0 | $0 |
| MLflow | EC2 $50+/월 | $0 | $0 | $0 |
| 서빙 | Docker $100+/월 | $32 | $360 | $500 |
| 학습 | 서버 포함 | $2 | $7 | $40 |
| 저장소 | 디스크 포함 | $0.25 | $1.25 | $5 |
| **합계** | **$2,300+/월** | **$36/월** | **$600/월** | **$640/월** |

> 핵심: AWS는 규모에 비례하여 비용이 증가하지만, On-Prem은 사용량과 무관하게 고정 비용 발생.
> 시나리오 1(중소형)에서는 On-Prem 대비 **98% 절감**.

## 비용 최적화 전략

| 전략 | 절감 효과 | 적용 대상 |
|------|----------|----------|
| SageMaker Spot 인스턴스 | ~70% | 학습, 배치 추론 |
| Lambda Provisioned Concurrency 최소화 | 콜드스타트 vs 비용 트레이드오프 | 서빙 |
| S3 Intelligent-Tiering | 자동 스토리지 클래스 전환 | 데이터 저장 |
| DynamoDB On-Demand | 프로비저닝 대비 소규모 저렴 | 피처 스토어 |
| Step Functions Express (단기 실행) | Standard 대비 저렴 | 파이프라인 |
| CloudWatch 로그 보존 기간 단축 | 저장 비용 | 모니터링 |

## Config 기반 자동 전환 요약

```yaml
# 이 설정만으로 규모별 최적 인프라 결정
serving:
  mode: auto
  auto_threshold: 150000000    # 월 1.5억 건 = Lambda↔ECS 손익분기

  feature_store: auto
  auto_feature_threshold: 5000000  # 유저 500만 = 메모리↔DynamoDB 전환

# CloudWatch 알림으로 임계값 접근 시 자동 알림
# → config 변경 → 재배포 → 인프라 전환 완료
```

### 온프레미스 비용

온프레미스에서는 Bedrock 비용이 발생하지 않는다. GPU 전력 비용(RTX 4070 ~200W)과 서버 유지비만 해당. 오픈소스 모델(Exaone, Qwen)은 라이선스 비용 없음(Apache 2.0). 에이전트 합의(WARN/FAIL 10건)는 로컬 GPU에서 ~45분, 추가 비용 $0.
