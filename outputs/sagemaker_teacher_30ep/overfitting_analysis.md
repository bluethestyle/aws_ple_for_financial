## 분석 결과

### 1. 과적합 유형: Loss-Metric Decoupling
- val_loss는 30ep까지 **단조 감소** (32.11 → 22.68) — 고전적 과적합(val_loss 상승) 아님
- 그러나 AUC는 epoch 10 (0.6726) 이후 정체/미세 하락 → epoch 30 (0.6687)
- NDCG@3는 epoch 10 (0.6976) 이후 **3%p 하락** → epoch 30 (0.6673)
- **Loss와 discriminative metric이 decouple**: regression task(MAE)가 loss를 계속 끌어내리지만, classification/ranking에는 기여 없거나 해로움

### 2. Cosine LR Restart의 영향
- T0=10 cosine restart 주기와 metric 변동이 정확히 일치:
  - Epoch 10 (1st cycle end): NDCG peak 0.6976
  - Epoch 11 (restart): 급락 0.6853 (-1.2%p)
  - Epoch 19 (2nd cycle near-end): 반등 0.7004
  - Epoch 20 (restart): 급락 0.6657 (-3.5%p)
  - Epoch 29 (3rd cycle near-end): 0.6540 (회복 실패)
- **LR restart가 cosine cycle 말기의 최적점을 이탈시킴**

### 3. 합성 데이터 한계
- AUC가 epoch 8~10에서 0.672로 빠르게 수렴 후 정체
- 합성 데이터의 패턴이 단순하여 10ep이면 학습 가능한 패턴을 전부 소진
- 추가 epoch은 노이즈를 학습할 뿐 일반화 개선 없음

### 4. 10ep vs 30ep 차이 요인
- Warmup (3 vs 5): 10ep는 epoch 4에서 이미 NDCG 0.7065 도달 (warmup 끝나자마자 최적점)
- num_workers (0 vs 2): batch 순서 차이 → 다른 local minimum (영향 미미)
- **best model 선택 기준**: 30ep의 best.pt는 val_loss 기준(epoch 30) 저장. AUC/NDCG 기준 best는 epoch 10.

### Teacher 모델 권장
- **증류용 teacher**: 10ep `joint_full` 모델 사용 (AUC 0.6724, NDCG@3 0.7065)
- 30ep epoch 10 checkpoint도 동등 수준이나, 10ep 로컬 모델이 NDCG에서 우위
- 30ep epoch 30은 val_loss는 최저이나 discriminative metric에서 열위

### 향후 개선
1. best model 선택 기준을 val_loss → composite metric (AUC + NDCG)으로 변경
2. Cosine restart 대신 cosine decay (restart 없음) 고려
3. Task type별 early stopping 분리: regression은 계속, classification은 10ep에서 freeze
